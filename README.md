Question # 2 Stack & design choices:
I used Python, Scikit-learn, Pandas, and Matplotlib because they are simple and efficient for machine learning and data visualization tasks like K-Means clustering.
Design Decisions
1.	Scatter Plot for Clusters
I used a scatter plot so users can visually see how data points are grouped into clusters instead of only reading numbers from a table. 
2.	Different Colors for Each Cluster
Each cluster has a separate color to make the groups easy to identify and compare in the graph.


Question #3Responsive & accessibility:
Responsive & Accessibility
On mobile screens like 360px, the layout adjusts and stacks properly so everything fits clearly.
On larger screens like 1440px, graphs and content use more space and look cleaner.
Accessibility Done
I used clear colors and good contrast so the graphs and text are easy to see.
Accessibility Not Added
I did not add full screen reader support because the project focus was mainly on the machine learning part.
